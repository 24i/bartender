# 24i Bartender

This repository contains code for the 24i bartender (both pi & application). The `app` folder contains the React Native frontend app and the `pi` folder contains all code handling the actual hardware control.

## Raspberry PI pinout

### Pump Pins
|Pin|Description|
|---|-----------|
|17|Pump 1|
|27|Pump 2|
|22|Pump 3|
|23|Pump 4|
|24|Pump 5|
|25|Pump 6|

### LED Pins
|Pin|Description|
|---|-----------|
|16|Red|
|20|Green|
|21|Blue|
